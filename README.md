# 💰 Expense Tracker (MERN Stack)

An advanced **Expense Tracker Web Application** built using the MERN stack that helps users manage, track, and analyze their financial transactions efficiently.

---

## 🚀 Features

* ✅ Add, Update, Delete Transactions (CRUD Operations)
* 📊 Data Visualization (Pie Chart & Bar Chart)
* 🌗 Dark Mode & Light Mode Toggle
* 📄 Download Expense Report (File Export)
* 💵 Real-time Balance Calculation
* 📜 Transaction History Tracking

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* CSS / Bootstrap

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

---

## 📊 Charts & Visualization

* **Pie Chart** → Shows category-wise expense distribution
* **Bar Chart** → Displays income vs expense comparison

---

## ⚙️ How It Works

1. User enters income or expense in the UI
2. Frontend sends request to backend (API)
3. Backend processes data and stores it in MongoDB
4. Updated data is sent back to frontend
5. UI updates dynamically with charts and balance

---

## 📄 Export Feature

Users can download their expense data as a file for:

* Offline access
* Record keeping
* Analysis

## 🎨 Theme Support

* 🌙 Dark Mode

## ▶️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/expense-tracker.git

# Navigate to project folder
cd expense-tracker

# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install

# Run backend
cd ../server
npm start

# Run frontend
cd ../client
npm start
```
